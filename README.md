# Airbnb Plus Program Analysis

This repository contains a statistics project in R examining the impact of the **Airbnb Plus Program** on platform performance across U.S. cities.

## Project overview

The analysis investigates whether the introduction of Airbnb Plus affected:

- **booking rates**
- **average listing reviews**

The project combines exploratory data analysis, hypothesis testing, and **Difference-in-Differences / fixed-effects regression** to compare the San Francisco pilot rollout with later implementations in other cities.

## Data

The cleaned dataset contains **14,643 observations** across **773 zip codes** in **11 U.S. cities**, covering the period from **August 2017 to October 2019**. Key variables include booking rate, average listing review, Airbnb search trend, entire-home ratio, and superhost ratio.

- hypothesis testing
